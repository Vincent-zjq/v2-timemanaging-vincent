# ⏰ 时间管理应用

一个功能完整的时间管理应用，帮助您高效管理日程和待办事项。

## ✨ 功能特性

### 📅 周日程表
- 完整的周视图时间表（周一至周日）
- 以每半小时为时间单位的时间轴
- 支持日程的添加、编辑、删除
- 支持拖拽调整日程位置
- 响应式设计，适配各种设备

### 📚 课表导入
- 支持导入Excel格式的课表文件
- 智能解析课程数据
- 自动识别时间和星期信息
- 提供导入预览功能

### 📋 日程管理
- 四种日程类型：学习类、运动类、娱乐类、事务类
- 每种类型有独特的颜色标识和图标
- 支持设置地点和详细描述
- 灵活的时长设置

### ✅ 待办事项
- 独立的待办事项管理界面
- 支持设置标题、描述和截止时间
- 三级优先级：高、中、低
- 完成状态标记
- 多种排序方式：按优先级、截止时间、创建时间
- 智能截止时间提醒

### 💾 数据持久化
- 所有数据自动保存到本地存储
- 支持数据导出为JSON文件
- 支持从备份文件恢复数据
- 一键清除所有数据

## 📖 使用说明

### 日程管理

1. **添加日程**
   - 点击"添加日程"按钮
   - 填写标题、选择类型、设置时间和时长
   - 可选填写地点和描述
   - 点击"添加"保存

2. **编辑日程**
   - 点击日程卡片打开编辑窗口
   - 修改需要更改的信息
   - 点击"保存"更新

3. **删除日程**
   - 点击日程卡片上的删除按钮
   - 确认删除操作

4. **拖拽调整**
   - 直接拖拽日程卡片到新的时间段

### 课表导入

1. 准备Excel文件，格式如下：
   ```
   时间    周一    周二    周三    周四    周五    周六    周日
   8:00    数学    英语    数学    英语    数学    -      -
   8:30    数学    英语    数学    英语    数学    -      -
   ```

2. 点击"导入课表"按钮
3. 选择Excel文件
4. 预览导入的课程信息
5. 点击"导入"完成

### 待办事项

1. **添加待办**
   - 点击"添加待办"按钮
   - 填写标题和描述
   - 设置优先级和截止时间
   - 点击"添加"保存

2. **完成待办**
   - 点击待办事项左侧的复选框
   - 已完成的待办会显示删除线

3. **编辑待办**
   - 点击编辑按钮（✏️）
   - 修改信息后保存

4. **删除待办**
   - 点击删除按钮（🗑️）
   - 确认删除操作

5. **筛选和排序**
   - 使用筛选按钮查看全部/进行中/已完成的待办
   - 使用排序下拉框按不同方式排序

### 数据管理

1. **导出数据**
   - 点击"导出数据"按钮
   - 自动下载JSON备份文件

2. **导入数据**
   - 点击"导入数据"按钮
   - 选择之前导出的JSON文件
   - 数据将自动恢复

3. **清除数据**
   - 点击"清除数据"按钮
   - 确认后删除所有数据
   - ⚠️ 此操作不可恢复

## 📱 响应式设计

应用完全响应式，支持以下设备：
- 桌面电脑（1920px+）
- 笔记本电脑（1024px - 1919px）
- 平板电脑（768px - 1023px）
- 手机（< 768px）

## 🔒 数据隐私

- 所有数据存储在浏览器本地
- 不会上传到任何服务器
- 完全离线可用
- 数据仅保存在您的设备上

## 📝 注意事项

1. **Excel导入格式**
   - 第一行必须是表头，包含星期名称
   - 第一列必须是时间
   - 时间格式支持：8:00、8:30等

2. **数据备份**
   - 建议定期导出数据备份
   - 清除浏览器数据会丢失所有信息

3. **浏览器兼容性**
   - 推荐使用最新版本的Chrome、Firefox、Edge或Safari
   - 需要支持ES6+的现代浏览器
